Introduction to matrix

A matrix is rectangular arrangement of elements.
e-g- $\A=[\table 1,2;3,4]$
A matrix having m rows and n columns is said to be of order m x n.
e-g- Order of matrix $A = [\table 1,2,3;4,5,6]$ is $ \2 $ x $ \3 $ as there is 2 rows and 3 columns.
An element of matrix is denoted by $ a_i_j $ where $\i$ the row number and $\j$ is the column number.
$$\table \a_11,a_12,a_13,...,a_1_n;a_21,a_22,a_23,...,a_2_n;a_31,a_32,a_33,...,a_3_n;...,...,...,...,...;...,...,...,...,...;a_m_1,a_m_2,a_m_3,...,a_m_n $$
$= [a_i_j]\where \i = 1,2,..,m;\j = 1,2,...n $
