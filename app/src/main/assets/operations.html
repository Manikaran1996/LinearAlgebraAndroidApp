Equality of matrices

Let A and B be two matrices. A and B are said to be equal if -

  1. order of both the matrices is same i.e., the number of rows and columns in A is equal to the number of rows and columns in B, and
  2. each element of one matrix is equal to the corresponding element of the other matrix i.e., $a_i_j = b_i_j$.
Hence, we can say that the two matrices are equal if and only if one is duplicate of the other.
If order of two matrices is same, then, they are said to be comparable.

Sum of matrices

Let A and B be two matrices of order m x n. Then, their sum $ A+B $ is defined as another matrix $ C $, such that any element of C is the sum of corresponding elements of A and B.
In similar manner, we can say $A-B$ is another matrix $C$, such that any element of C is the difference of the corresponding elements of A and B.
$C = A ± B = [ a_i_j ± b_i_j]$

e-g- $ A = [\table 1,8,9;2,3,4] $ and $ B =[\table 0,3,4;5,2,3] $
Here, order of both A and B is 2 x 3. Hence they can be added.
$A+B = [\table 1+0,8+3,9+4;2+5,3+2,4+3]$
$ = [\table 1,11,13;7,5,7]$
and $A-B = [\table 1-0,8-3,9-4;2-5,3-2,4-3]$
$ = [\table \1,5,5;-3,1,1]$

Properties of com.har_asha.linearalgebra.Matrix Addition

  1. com.har_asha.linearalgebra.Matrix Addition is commutative $$A+B = B+A$$
  2. com.har_asha.linearalgebra.Matrix Addition is associative $$(A+B)+C = A+(B+C)$$

Scalar Multiplication of a matrix

Let A be a given matrix and $k$ be any scalar number real or complex, then, $kA$ is the matrix in which each element of matrix A is multiplied by $k$ and is called scalar multiplication of $A$ with $k$.
e-g- If $A = [\table 2,3,1;5,2,4]$
then, $\4A = [\table 4.2,4.3,4.1;4.5,4.2,4.4]$
or $\4A = [\table 8,12,4;20,8,16]$
