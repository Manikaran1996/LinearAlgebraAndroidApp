Example 1

Let $A = [\table 1,2,3]$ and $B = [\table 4;5;6]$
Order of A = 1 x 3.
Order of B = 3 x 1.
As the number of columns of A = number of rows of B.
So, matrix AB can be obtained.
Order of AB will be 1 x 1 .
$AB = [\table 1\x4+2\x5+3\x6]$ = $[\table 32]$.
Similarly, matrix BA can also be obtained.
Order of BA will be 3 x 3.
$BA = [\table 4;5;6] [\table 1,2,3]$
$=[\table 4.1,4.2,4.3;5.1,5.2,5.3;6.1,6.2,6.3]$
$=[\table 4,8,12;5,10,15;6,12,18]$.

Note : We can observe from this example that AB ≠ BA .
