Product of two Matrices

For two matrices A and B, the product matrix AB can be obtained if the number of columns in A is equal to the number of rows in B.

Rule of Multiplication

Let $A = [\table a,b;c,d;e,f]$ , $B = [\table u,v,w;x,y,z]$
Order of A = 3 x 2
Order of B = 2 x 3
As, number of columns in matrix A = number of rows in matrix B.
So, matrix AB can be found.
AB = $[\table a,b;c,d;e,f]$ $[\table u,v,w;x,y,z]$
$[\table au+bx,av+by,aw+bz;cu+dx,cv+dy,cw+dz;eu+fx,ev+fy,ew+fz]$
Element $c_i _j$ of C = sum of the products of corresponding elements of the $ith$ row of A with those of $jth$ column of B.
Let A and B be two matrices of order p x q and q x r and C be the product of A and B, then, $$c_i_j = ∑↙{i=1}↖q a_i_k*b_k_j $$

Order of matrix C = p x r.
