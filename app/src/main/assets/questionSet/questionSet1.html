Q1. $\cos\A [\table \cos\A ,\sin\A ;\-\sin\A,\cos\A]$ + $\sin\A [\table \sin\A ,\-\cos\A ;\cos\A ,\sin\A]$ is equal to

A. $[\table \1,0;0,1]$

B. $[\table \0,1;1,0]$

C. $[\table \0,0;1,1]$

D. None of these

Show Answer

Ans. $\cos\A [\table \cos\A ,\sin\A ;\-\sin\A,\cos\A]$ + $\sin\A [\table \sin\A ,\-\cos\A ;\cos\A ,\sin\A]$
= $[\table \cos^2A,cos\A\sin\A;-\cosA\sinA,\cos^2A]$ + $[\table \sin^2A,-cos\A\sin\A;sin\A\cos\A,\sin^2A]$
= $[\table \cos^2A\+\sin^2A,\0;0,\cos^2A\+\sin^2A]$
= $[\table \1,0;0,1]$

Q2. If $A = [\table x,0;1,1]$ , $ B = [\table \1,0;5,1]$ whenever $ A^2 = B $, then the value of $x$ is

A. 1

B. -1

C. 4

D. no real value of x

Show Answer

Ans. $A^2=[\table x,0;1,1]$ $[\table x,0;1,1]$
= $[\table x^2+0,0+0;x+1,0+1]$
⇒ $[\table x^2,0;x+1,1]$ = $[\table \1,0;5,1]$
⇒ $x+1 = 5$ , $x=4$
Also , $x^2 = 1$ ⇒ $x = ±1$
Hence, No solution exists.

Q5. If $ A = [\table 0,1;1,0]$ , then $ A^4 $ =

A. $[\table 1,0;0,1]$

B. $[\table 1,1;0,0]$

C. $[\table 0,0;1,1]$

D. $[\table 0,1;1,0]$

Show Answer

Ans. $A^4 = A^2.A^2$
$A^2=[\table 0,1;1,0]$ $[\table 0,1;1,0]$
$ = [\table 0+1,0+0;0+0,1+0]$
= $[\table 1,0;0,1]$
$A^4 = [\table 1,0;0,1]$ $[\table 1,0;0,1]$
= $[\table 1,0;0,1]$
