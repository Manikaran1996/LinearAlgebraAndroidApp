Q3. If A is any m x n matrix such that AB and BA are both defined, then, N is a matrix of type

A. m x n

B. n x m

C. m x m

D. n x n

Show Answer

Ans. AB is defined if no. of columns in A = no. of rows in B.
So, no. of rows in B should be 'n'.
For BA to be defined, no. of cols in B = no. of rows in A.
So, no. of cols in B should be 'm'.
Hence, B is a matrix of type 'n x m'.

Q4. If $A= [\table 1,-2,3;-4,2,5]$, $B= [\table 2,3;4,5;2,1]$, then

A. AB , BA exist and are equal.

B. AB , BA exist but not equal.

C. AB exists and BA does not exist.

D. AB does not exist and BA exists.

Show Answer

Ans. Order of A = 2 x 3.
Order of B = 3 x 2
So, AB and BA both exist.
Order of AB = 2 x 2.
Order of BA = 3 x 3.
Hence,they can not be equal.

Q5. The order of $[\table x,y,z]$ $[\table a,h,g;h,b,f;g,f,c]$ $[\table x;y;z]$ is

A. 3 x 1

B. 1 x 3

C. 1 x 1

D. 3 x 3

Show Answer

Ans. Let $A=[\table x,y,z]$
$B=[\table a,h,g;h,b,f;g,f,c]$
$C=[\table x;y;z]$
Order of A = 1 x 3
Order of B = 3 x 3
Order of C = 3 x 1
Order of AB = 1 x 3
Order of ABC = 1 x 1
