Q1. The number of all possible matrices of order 3 x 3 with each entry 0 or 1 is

A. 27

B. 18

C. 81

D. 512

Show Answer

Ans. No. of elements in a matrix of order 3 x 3 = 9
Each entry is either 0 or 1.
So, total no. of matrices = $ 2^9 $ = $512$.

Q2. The matrices $[\table 2x+y,3x;5x-y,7y-4x]$ = $[\table 7,6;7,13]$ if

A. x = 2 , y = 3

B. x = 3 , y = 1

C. x = 1 , y = 5

D. x = 3 , y = 2

Show Answer

Ans. As $[\table 2x+y,3x;5x-y,7y-4x]$ = $[\table 7,6;7,13]$
So, $2x+y=7$ ... $(i)$
$3x = 6$ ... $(ii)$
$5x-y=7$ ... $(iii)$
$7y-4x=13$ ... $(iv)$
from $(ii)$ ,
$x=2$
putting $x=2$ in $(i)$ ,
we get, $2(2)+y=7$
⇒ $y=3$

Q3. Let P and Q be two matrices of order 2 x n and 2 x p respectively. If $n=p$, then order of the matrix 4P-3Q is

A. p x 2

B. 2 x n

C. n x 3

D. p x n

Show Answer

Ans. As, order of P = 2 x n
So, order of 4P is also 2 x n.
Order of Q = 2 x p.
So , order of 3Q is also 2 x p.
As $n=p$, So, these two matrices can be subtracted.
Order of 4P - 3Q = 2 x p or 2 x n.

Q4. If A and B are symmetric matrices of the same order, then, AB-BA is a

A. Skew-symmetric matrix

B. Symmetric matrix

C. Zero matrix

D. Identity matrix

Show Answer

Ans. $(AB)^T = B^T A^T$
As, A and B are symmetric matrices
So, $A=A^T$ and $B=B^T$
⇒ $(AB)^T = BA$.
Similarly, $(BA)^T = A^T B^T$ = $AB$.
$ ⇒(AB-BA)^T = (AB)^T - (BA)^T $ $= BA-AB $ = $-(AB-BA) $
⇒ AB-BA is a skew symmetric matrix.

Q5. If $A = [\table \cos\x,-\sin\x;\sin\x,\cos\x]$ and $A+A^T = I$, then value of $x$ is

A. $π/6$

B. $π$

C. $π/3$

D. $ (3π)/2$

Show Answer

Ans. $A^T = [\table \cos\x,-\sin\x;\sin\x,\cos\x]^T$
$ = [\table \cos\x,\sin\x;-\sin\x,\cos\x]$
$A+A^T $ = $ [\table \cos\x,-\sin\x;\sin\x,\cos\x] + [\table \cos\x,\sin\x;-\sin\x,\cos\x]$
= $[\table 2\cos\x,0;0,2\cos\x]$
= $[\table 1,0;0,1]$
⇒ $2 \cos x = 1 $
$\cos x = 1/2 $
⇒ $ x = π/3$
