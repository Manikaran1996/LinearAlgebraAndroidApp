Q1. If A is a square matrix such that $A^2 = A$, then $(\I + \A)^3 - 7\A$ is equal to

A. A

B. I-A

C. I

D. 3A

Show Answer

Ans. $(I+A)^3 $ = $I^3 + A^3 + 3A^2 I + 3IA^2 $
= $I + A^2 A + 3AI + 3IA$
= $ I + A.A + 3A + 3A$
= $I + A + 6A$
= $I+7A$
⇒ $(I+A)^3 -7A $ = $ I + 7A - 7A $ = $ I $

Q2. If $|\table x , 5; 3 , 2x|$ = $|\table 4,7;1,6|$, then, value of x is

A. 4

B. ± 4

C. -4

D. ± 1

Show Answer

Ans. $|\table x , 5; 3 , 2x|$ = $|\table 4,7;1,6|$
or   $2x^2 -15 = 17$
or   $2x^2 = 32$
or   $x^2 = 16$
or   $x = ±4 $

Q3. The value of the determinant $|\table 2,6,8;5,12,19;1,3,4|$ is

A. 0

B. 1

C. 12

D. 2

Show Answer

Ans. $|\table 2,6,8;5,12,19;1,3,4|$ = $2 |\table 1,3,4;5,12,19;1,3,4|$
As R1 and R3 are same
So, $|\table 2,6,8;5,12,19;1,3,4| = 0$

Q4. Let A be a square matrix of order 3 x 3. Then $|kA|$ is equal to

A. $k|A|$

B. $k^3 |A|$

C. $k^2 |A|$

D. $3k|A|$

Show Answer

Ans. If A is a square matrix of order 'm'
then, $|kA| = k^m |A|$
