Symmetric Matrices

A square matrix A = $[a_i_j]$ is called symmetric if A = AT i.e., matrix A is equal to its transpose.

In a symmetric matrix for all values of i and j, $a_i_j = a_j_i$ i.e., every i-jth element = j-ith element.
e-g- $[\table a,h,g;h,b,f;g,f,c]_3_\x_3$

A + AT is always a SYMMETRIC MATRIX.

Skew Symmetric Matrices

A square matrix A = $[a_i_j]$ is called skew symmetric if A = -AT i.e., matrix A is equal to negative of its transpose.

In a skew symmetric matrix for all values of i and j, $a_i_j = -a_j_i$ i.e., its i-jth element is negative of j-ith element.
e-g- $[\table 0,h,g;-h,0,f;-g,-f,0]_3_\x_3$

A - AT is always a SKEW SYMMETRIC MATRIX.

The diagonal elements of a skew symmetric matrix are ZERO.

Note : A matrix can be expressed as the sum of a symmetric and a skew symmetric matrix.
$ \A = 1/2 (\A + \A^T) + 1/2 (\A - \A^T)$
