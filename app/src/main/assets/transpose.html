• If A be a given matrix of the order m x n, then, the matrix obtained by interchanging the rows and columns of the matrix i.e., by changing the rows of A into columns and columns of A into rows is called Transpose of matrix A.
  • Transpose of matrix A is denoted by A' or AT.
  • If the order of matrix A is m x n then, the order of A' (transpose of A) is n x m.

e-g- $$\A = (\table 30,14;12,11;15,19)$$ $$\A' = (\table 30,12,15;14,11,19)$$

Properties of Transpose

  1. (A ') ' = A
  2. ( kA )' = k A'
  3. ( A + B )' = A' + B'
  4. (AB)' = B'A'
  5. (ABC)' = C'B'A'
