Types of matrix

Column Matrix
A matrix is said to be a column matrix if it has only one column.
e-g- $ A = \id "light" [\table \id "element" 1;\id "element" 2;\id "element" 3]$
Order of the column matrix is m x 1.
Row Matrix
A matrix is said to be a row matrix if it has only one row.
e-g- $ A = [\table 1,2,3,4]$
Order of the row matrix is 1 x n.
Square Matrix
A matrix in which number of rows are equal to the number of columns is said to be a square matrix.
A square matrix of order 'n' has n  columns and n rows.
e-g- $ A = [\table \0,1,2;2,3,4;5,6,7]$
Diagonal Matrix
A square matrix is said to be diagonal matrix if all its non diagonal elements are zero.
e-g- $ A = [\table 5]$ $ B =[\table \2,0;0,-1]$
Scalar Matrix
A diagonal matrix is said to be a scalar matrix if its diagonal elements are equal.
e-g- $A = [\table -2,0;0,-2]$ , $B = [\table -3,0,0;0,-3,0;0,0,-3]$ are scalar matrices of order 2 and 3.
Identity Matrix
A square matrix in which elements in the diagonal are all 1 and rest are all zero is called an identity matrix.
The identity matrix of order n is denoted by $I_n$
e-g- $A = [\table \1]$ , $B = [\table 1,0;0,1]$ , $C = [\table 1,0,0;0,1,0;0,0,1]$ are Identity matrices of order 1 , 2 and 3.
Zero Matrix
A matrix is said to be a zero matrix or null matrix if all its elements are zero.
e-g- $A=[\table \0]$ , $B=[\table 0,0]$ , $C = [\table \0,0,0;0,0,0]$ are all zero matrices.
A zero matrix is denoted by $O$.
